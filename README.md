## **Featured Projects**

### **[MoleculeGNN](https://github.com/SamarthPyati/MoleculeGNN)** - Graph Neural Networks for Molecular Property Prediction
Applied deep learning to chemistry by building a Graph Convolutional Network (GCN) to predict molecular toxicity, solubility, and physical properties from SMILES strings, representing molecules as graphs with atoms as nodes and bonds as edges
* **Tech Stack:** PyTorch-Geometric, scikit-learn, RDKit, MoleculeNet, Streamlit
* **Achievements:** Superior performance over traditional fixed-length fingerprint-based ML methods by directly learning from molecular graph structure, capturing complex chemical patterns including functional groups and ring systems

### **[DocSense](https://github.com/SamarthPyati/DocSense)** - Lightning-Fast Local Document Search
Locally deployable document search engine in Rust using BM25 scoring and indexing for rapid retrieval on large text corpora
* **Tech Stack:** Rust, libpoppler
* **Performance:** 2-5ms query response time on 50k document corpus with 85% recall at k=10
* **Features:** Recursive document indexing with custom lexer for fast tokenization, persistent JSON index storage, served via local HTTP web interface

### **[iDetect](https://github.com/SamarthPyati/iDetect)** - Automated Facial Recognition Attendance
Smart attendance system with real-time facial recognition and admin dashboard
* **Tech Stack:** Python, Streamlit, dlib, PyTorch
* **Impact:** Streamlining attendance tracking with 99%+ accuracy

### **[Seam Carving](https://github.com/SamarthPyati/seam-carving)** - Content-Aware Image Resizing
Content-aware image resizing algorithm that selectively removes or inserts pixel seams to intelligently resize images while preserving important visual content without machine learning
* **Tech Stack:** C, OpenMP, LLDB
* **Implementation:** Efficient parallel processing using OpenMP to optimize computationally intensive energy map calculation and seam removal operations
* **Based on:** [ACM SIGGRAPH paper "Seam carving for content-aware image resizing"](https://www.cs.princeton.edu/courses/archive/fall10/cos429/papers/seamcarving.pdf) with custom build system using [`nob.h`](https://github.com/tsoding/nob.h)

### **[sp_arena](https://github.com/SamarthPyati/sp_arena)** - High-Performance Memory Arena Allocator
Lightweight arena-based memory allocator library with efficient memory management through large block allocation and subdivision, minimizing overhead and fragmentation
* **Tech Stack:** C, Valgrind, LLDB
* **Features:** Multiple allocation methods (standard, aligned, zero-initialized), temporary arena scopes for scoped memory management, optional thread safety support
* **Capabilities:** Customizable configuration for block size, alignment, custom allocators, and comprehensive error handling with memory utilization statistics

### **[grive](https://github.com/SamarthPyati/grive)** - OpenGL Text Editor
Dramatic keystroke-responsive text editor built from scratch with OpenGL
* **Tech Stack:** C, OpenGL, SDL2

### **[Loan Eligibility Predictor](https://github.com/SamarthPyati/Loan-Predictor)** - ML-Powered Loan Approval
Intelligent loan approval system using Random Forest and Neural Networks
* **Tech Stack:** Python, Scikit-learn, TensorFlow, Streamlit, Plotly

### **[Blogify](https://github.com/SamarthPyati/Blogify)** - Full-Stack Blogging Platform
Complete blogging platform with authentication and database integration
